# بوت محرم (Muharram Bot)

بوت تيليجرام متكامل لإدارة حسابات متعددة، نشر تلقائي، انضمام آلي، جلب روابط، مع لوحة تحكم متطورة.

## المميزات
- إدارة الأرقام (إضافة، عرض، فحص، حذف) مع تشفير الجلسات
- محرك نشر ذكي مع إمكانية الإيقاف المؤقت والاستئناف
- محرك انضمام آلي مع دعم الانضمام الذكي
- جلب روابط تيليجرام وواتساب من المحادثات
- نظام اشتراكات بأكواد تفعيل
- لوحة أدمن متكاملة مع إحصائيات
- مساعد القائد لإدارة الأكواد
- دعم البروكسي (SOCKS5)
- نظام مرايا (Mirror Token) للطوارئ
- نسخ احتياطي تلقائي لقاعدة البيانات إلى قناة خاصة
- تحديث سهل عبر Git
